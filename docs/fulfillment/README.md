# Fulfillment 文档

这个目录存放履约相关文档，包括：

- 订单备注区
- 操作审计日志
- 发货 SOP
- 列表级快捷履约动作
- 其他后台履约增强项

## 命名约定

新功能文档优先使用以下命名：

- `<feature>-design.md`
- `<feature>-api-contract.md`
- `<feature>-test-plan.md`
- `<feature>-migration.md`

如果功能还在草案阶段，可先只建一份：

- `<feature>-design.md`
