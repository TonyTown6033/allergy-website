## 1. 验收范围

- 管理员查看备注列表
- 管理员新增备注
- 管理员查看操作日志
- 关键履约动作自动写入日志

## 2. 后端测试

- 获取备注列表成功
- 新增备注成功
- 非管理员不可写备注
- 获取日志列表成功
- 发货、样本签收、开始检测、发布报告等动作会写入日志

## 3. 前端联调

- 订单详情页显示备注区
- 订单详情页显示操作日志区
- 新增备注后页面可刷新看到结果
- 日志按时间倒序展示

## 4. 回归重点

- 现有履约动作不被备注功能破坏
- 现有时间线展示不受影响
