# 发货 SOP 文档骨架

日期：2026-04-15

状态：

- outline

## 1. 目标

为“已支付订单寄送采样盒”建立一份可交付给运营使用的标准操作说明。

## 2. 适用范围

- 过敏原检测订单
- 后台管理员或运营人员
- 采样盒首次寄送与补寄场景

## 3. 建议章节

### 3.1 发货前检查

- 订单支付状态
- 收件信息完整性
- 是否存在特殊备注
- 是否需要补寄或改址确认

### 3.2 包装检查

- 采样盒编码
- 随盒说明物料
- 防错核对项

### 3.3 系统录入

- `kit_code`
- `outbound_carrier`
- `outbound_tracking_no`
- `outbound_shipped_at`

### 3.4 发货后动作

- 更新系统状态
- 核对时间线写入
- 必要时通知用户

### 3.5 异常处理

- 地址错误
- 物流退回
- 补寄
- 重复发货

- 发货时限要求
