# Auth 文档

这个目录存放会员账号体系相关文档，包括：

- 注册
- 登录
- 找回密码
- 会员状态与准入校验
- 旧会员清理与切流

## 命名约定

新功能文档优先使用以下命名：

- `<feature>-design.md`
- `<feature>-api-contract.md`
- `<feature>-test-plan.md`
- `<feature>-migration.md`

如果功能还在草案阶段，可先只建：

- `<feature>-design.md`
