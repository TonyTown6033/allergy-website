## 1. 验收范围

- 管理员发起退款
- 退款状态回写
- 退款受订单履约阶段约束

## 2. 后端测试

- 已支付未发货订单可发起退款
- 已发货订单不可直接发起退款
- 退款动作会写入操作人、时间、备注
- 退款结果会回写订单状态和支付状态

## 3. 前端联调

- 管理端详情页可见退款入口
- 不满足条件时退款按钮禁用或提示原因
- 退款完成后详情页显示退款结果

## 4. 回归重点

- 不影响正常支付成功订单的履约链路
- 不影响对账页读取支付字段
